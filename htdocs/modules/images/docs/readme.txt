Read Me First
-------------------

Description
------------
The Images Extension is for managing images that can be used in the system.

Requirements
 _____________________________________________________________________

- PHP version >= 5.3
- XOOPS 2.6.0+

Install/uninstall
------------------
No special measures necessary, follow the standard installation process & extract the images folder into the modules/ directory. Install the module through Admin -> Extensions administration.

Detailed instructions on installing modules and extensions are available in the XOOPS Operations Manual (http://goo.gl/adT2i)

Operating instructions
------------------------
To set up this module you need to:

i)   Create Categories to organize your images. At least one category is required.

ii)  Configure your preferences for the module (see "Preferences")

iii) Add images to suit your needs.
