Read Me First
-------------------

Description
------------
The Smilies Extension is for managing character interpretation and images of smilies.

Requirements
 _____________________________________________________________________

- PHP version >= 5.3
- XOOPS 2.6.0+

Install/uninstall
------------------
No special measures necessary, follow the standard installation process & extract the smilies folder into the modules/ directory. Install the module through Admin -> Extensions administration.

Detailed instructions on installing modules and extensions are available in the XOOPS Operations Manual (http://goo.gl/adT2i)

Operating instructions
------------------------
To set up this extension you need to:

i)   Verify or customize the smilies in the Smilies manager tab.

ii)  Configure your preferences for the module (see "Preferences")
