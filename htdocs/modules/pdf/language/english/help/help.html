Description

The pdf extension establishes a service provdier for the HtmlToPdf service. This service allows modules to add PDF (Portable Document Format) output with minimal effort. Modules can also customize the Pdf in several ways, including adding metadata, such as author, title and keywords,

Installation

The Pdf extension installs like any other XOOPS extension. Only one HtmlToPdf supplier can be used in XOOPS system at a time.

Preferences

Pdf's preferences allow you to establish the default page parameters, orientation, size, fonts and more. Pdf uses the TCPDF package. For an in depth discussion of some of the options available in preferences, see this page:
http://www.tcpdf.org/
